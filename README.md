# Login Page Automation Testing 🚀

Automated login form testing using **Playwright** and **JavaScript**

## 🛠️ Tech Stack
- Playwright
- JavaScript
- Node.js

## 🧪 Test Cases
| Test | Description | Status |
|------|-------------|--------|
| Valid Login | Correct username & password | ✅ Pass |
| Invalid Login | Wrong credentials | ✅ Pass |
| Empty Fields | Both fields empty | ✅ Pass |
| Empty Password | Only username filled | ✅ Pass |
| Empty Username | Only password filled | ✅ Pass |

- Failed: 0 ❌

## ▶️ How to Run
npm install
npx playwright install
npx playwright test
npx playwright show-report
